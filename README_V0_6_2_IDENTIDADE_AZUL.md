# ERP Duo Print 3D — Versão 0.6.2

## Identidade visual azul

Esta versão remove a tonalidade roxa/lilás herdada do Material Design e
padroniza o aplicativo na paleta da Duo Print 3D.

### Paleta aplicada

- Azul principal: `#1260DC`
- Azul escuro: `#0B4DB3`
- Azul secundário: `#3B82F6`
- Azul claro: `#8AB4FF`
- Fundo: `#090D14`
- Cards: `#111827`
- Texto principal: `#F8FAFC`
- Texto secundário: `#B8C4D6`

### Componentes padronizados

- textos e títulos;
- botões;
- ícones;
- links;
- abas;
- menu inferior;
- cards;
- campos de formulário;
- barras de progresso;
- chips;
- caixas de seleção;
- botões de alternância;
- calendários;
- mensagens;
- menus;
- indicadores selecionados.

## Instalação

Substitua no projeto:

- `lib`
- `pubspec.yaml`

Depois execute:

```cmd
flutter clean
flutter pub get
flutter analyze
flutter run
```
