# ERP DUO PRINT 3D — Sprint 03A

## Implementado
- motor de custos;
- seleção de projeto e filamento;
- quantidade;
- material, energia, máquina, mão de obra, embalagem e falhas;
- margem e lucro;
- preço de venda calculado;
- acesso pelo menu Módulos.

## Instalação
Esta entrega é incremental.

1. Faça backup do projeto.
2. Extraia o ZIP.
3. Copie a pasta `lib` sobre a pasta `lib` atual.
4. Confirme a substituição.
5. Execute:

```cmd
flutter clean
flutter pub get
flutter analyze
flutter run
```
